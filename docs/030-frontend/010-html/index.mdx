# HTML

HTML (Hypertext Markup Language) to standardowy język znaczników używany do tworzenia stron internetowych. Jest to podstawowy budulec większości stron internetowych i stanowi fundament struktury i zawartości witryn internetowych. HTML pozwala na określenie struktury dokumentu, jak również umożliwia osadzanie różnych rodzajów mediów, takich jak tekst, obrazy, dźwięki i linki. Oto kilka kluczowych informacji na temat HTML:

1. **Znaczniki HTML**: HTML używa znaczników (tagów), które są otwierane za pomocą znaku "<" i zamykane za pomocą znaku ">". Przykładem jest znacznik `<html>`, który oznacza początek dokumentu HTML. Wszystkie elementy na stronie internetowej są opisane za pomocą znaczników.

2. **Struktura dokumentu**: Dokument HTML składa się z kilku głównych sekcji, takich jak `<html>`, `<head>` (nagłówek) i `<body>` (ciało dokumentu). W `<head>` znajdują się metadane, takie jak tytuł strony i odnośniki do arkuszy stylów i innych zasobów. W `<body>` znajduje się właściwa zawartość strony widoczna dla użytkownika.

3. **Elementy tekstowe**: HTML pozwala na umieszczanie tekstu na stronie za pomocą znaczników, takich jak `<p>` (paragraf), `<h1>`, `<h2>` itd. (nagłówki różnych poziomów), `<ul>` i `<li>` (lista wypunktowana), `<ol>` i `<li>` (lista numerowana), `<a>` (link), `<strong>` (pogrubienie tekstu) i wiele innych.

4. **Obrazy i multimedia**: Za pomocą HTML można osadzać obrazy i multimedia na stronie internetowej za pomocą znacznika `<img>` dla obrazów, `<audio>` dla dźwięku i `<video>` dla filmów. Zazwyczaj te elementy wymagają dodatkowych atrybutów, takich jak źródło (src) i opis (alt).

5. **Linki**: HTML umożliwia tworzenie linków do innych stron internetowych za pomocą znacznika `<a>`. Linki te mogą być tekstowe lub graficzne i pozwalają na nawigację między różnymi stronami internetowymi.

6. **Formularze**: Formularze HTML pozwalają użytkownikom na wprowadzanie danych, takich jak tekst, liczby, wybory z listy rozwijanej itp. Za pomocą znaczników takich jak `<form>`, `<input>`, `<textarea>` i `<select>` można tworzyć różne rodzaje formularzy.

7. **Komentarze**: W HTML można dodawać komentarze, które nie są widoczne dla użytkowników, ale mogą być przydatne dla programistów lub projektantów do dokumentowania kodu. Komentarze są otaczane znacznikiem `<!--` i `-->`.

8. **Wersje HTML**: HTML jest stale rozwijanym językiem, i istnieje wiele jego wersji. Obecnie najnowszą wersją jest HTML5, która wprowadza wiele nowych funkcji i usprawnień.

9. **Arkusze stylów CSS**: Chociaż HTML jest używane do określenia struktury i zawartości strony, wygląd i styl strony są kontrolowane za pomocą arkuszy stylów CSS (Cascading Style Sheets).

10. **Interakcja z JavaScript**: HTML i JavaScript współpracują ze sobą, pozwalając na tworzenie interaktywnych elementów i funkcjonalności na stronie internetowej.

HTML jest podstawowym narzędziem dla każdego webmastera i programisty, który tworzy strony internetowe. Poznanie podstaw HTML jest ważne dla każdej osoby zainteresowanej tworzeniem treści i stron internetowych.
