# SVG

SVG (Scalable Vector Graphics) to format grafiki wektorowej używany do opisu grafiki za pomocą wektorów, co oznacza, że obrazy są definiowane za pomocą matematycznych kształtów i krzywych, a nie pikseli, co jest typowe dla grafiki rastrowej. Format SVG ma wiele zastosowań, zarówno na stronach internetowych, jak i poza nimi. Oto kilka kluczowych informacji na temat SVG:

1. **Skalowalność**: Jednym z głównych zalet SVG jest jego skalowalność. Obrazy SVG mogą być dowolnie powiększane lub zmniejszane bez utraty jakości i ostrości. To oznacza, że są idealne do tworzenia grafiki dla różnych urządzeń i rozmiarów ekranów, w tym na stronach internetowych, w aplikacjach mobilnych i w druku.

2. **Struktura XML**: SVG jest oparty na XML (eXtensible Markup Language), co oznacza, że jest strukturą tekstu i można go edytować za pomocą tekstu. Każdy element grafiki SVG jest opisany jako znacznik XML, co czyni go przyjaznym dla maszyn i programistów.

3. **Podstawowe elementy**: W SVG można używać różnych podstawowych elementów, takich jak linie, prostokąty, koła, elipsy, łuki, wielokąty i tekst. Każdy z tych elementów jest opisany za pomocą odpowiednich znaczników XML.

4. **Atrybuty**: Elementy SVG mają atrybuty, które kontrolują ich właściwości, takie jak kolor wypełnienia, kolor obramowania, grubość linii itp. Na przykład atrybut `fill` kontroluje kolor wypełnienia, a `stroke` kontroluje kolor obramowania.

5. **Efekty i gradienty**: SVG pozwala na dodawanie różnych efektów do grafiki, takich jak cienie, przejścia kolorów, gradienty i przezroczystość.

6. **Animacje**: SVG umożliwia tworzenie animacji za pomocą atrybutów i elementów animacyjnych. Można tworzyć animowane efekty, takie jak obracanie, powiększanie i przesuwanie.

7. **Interakcja z JavaScript**: SVG można również łączyć z JavaScript, co umożliwia tworzenie interaktywnych grafik i wykresów. JavaScript może manipulować i reagować na elementy SVG.

8. **Obszar rysowania**: Element `<svg>` definiuje obszar, na którym będzie rysowana grafika SVG. Wewnątrz tego elementu można dodawać wszystkie inne elementy grafiki.

9. **Ikony i wykresy**: SVG jest często stosowany do tworzenia ikon, symboli, logo oraz wykresów i grafik danych na stronach internetowych.

10. **Współpraca z innymi technologiami**: SVG może być używane w połączeniu z innymi technologiami webowymi, takimi jak HTML, CSS i JavaScript, aby tworzyć interaktywne strony internetowe i aplikacje.

SVG jest szeroko stosowany w projektowaniu stron internetowych, zwłaszcza w przypadku elementów graficznych, które muszą zachować jakość na różnych urządzeniach. Jest również używany w projektowaniu grafiki wektorowej do druku, tworzenia ikon, animacji i innych zastosowań, gdzie jakość i skalowalność są ważne.
