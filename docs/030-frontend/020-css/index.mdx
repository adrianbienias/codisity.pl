# CSS

CSS (Cascading Style Sheets) to język używany do opisywania wyglądu i prezentacji stron internetowych lub innych dokumentów napisanych w HTML lub XML. CSS umożliwia projektantom kontrolowanie kolorów, czcionek, marginesów, odstępów między elementami, tła i wielu innych aspektów wizualnych treści na stronie internetowej. Oto kilka kluczowych informacji na temat CSS:

1. **Zastosowanie do HTML**: CSS jest używane w połączeniu z dokumentami HTML do określania stylu i wyglądu elementów na stronie internetowej. Związane z HTML dokumenty CSS znajdują się zazwyczaj w oddzielnych plikach lub mogą być osadzone bezpośrednio wewnątrz dokumentu HTML.

2. **Selektory**: W CSS używa się selektorów, które określają, które elementy na stronie mają być stylizowane. Na przykład selektor `h1` zostanie dopasowany do wszystkich nagłówków pierwszego poziomu na stronie, a selektor `#id-elementu` odnosi się do elementu o określonym identyfikatorze.

3. **Właściwości i wartości**: CSS definiuje właściwości (np. `color`, `font-size`, `margin`, `padding`, `background-color`) i przypisuje im wartości (np. `red`, `16px`, `10px`, `#f0f0f0`). Wartości te kontrolują wygląd elementów na stronie.

4. **Reguły CSS**: Reguły CSS składają się z selektora, nawiasów klamrowych `{}`, a wewnątrz nawiasów umieszczone są deklaracje właściwości i wartości. Przykład:

   ```css
   h1 {
     color: blue;
     font-size: 24px;
   }
   ```

   Powyższa reguła CSS zmienia kolor i rozmiar czcionki dla wszystkich nagłówków pierwszego poziomu (`<h1>`) na stronie.

5. **Kaskadowość**: C w CSS oznacza "cascading" (kaskadowanie), co oznacza, że stylizacja może być dziedziczona z jednego elementu na inny. W przypadku konfliktu stylów, kaskadowość pomaga określić, który styl ma pierwszeństwo.

6. **Selektory zagnieżdżone**: CSS pozwala na tworzenie bardziej precyzyjnych selektorów poprzez zagnieżdżanie. Na przykład można użyć selektora potomka, aby wybrać elementy, które są potomkami innego elementu.

7. **Arkusz stylów zewnętrzny**: Wielokrotnie używane style CSS mogą być przechowywane w oddzielnym arkuszu stylów zewnętrznych (plik `.css`), który jest dołączany do dokumentu HTML za pomocą tagu `<link>` w sekcji `<head>`.

8. **Inline CSS**: Można również stosować style CSS bezpośrednio wewnątrz elementów HTML, używając atrybutu `style`. Na przykład:

   ```html
   <p style="color: green; font-size: 18px;">
     Ten tekst jest zielony i ma rozmiar 18px.
   </p>
   ```

9. **Responsywność**: CSS jest często używane do tworzenia responsywnych stron internetowych, które dostosowują się do różnych rozmiarów ekranów i urządzeń.

10. **Transformacje i animacje**: CSS umożliwia także tworzenie efektów transformacji (np. obracanie, skalowanie) oraz animacji, które nadają stronie interaktywny charakter.

CSS jest kluczowym narzędziem dla projektantów i deweloperów webowych, ponieważ pozwala na kontrolę wyglądu i stylu stron internetowych. Dzięki CSS można tworzyć atrakcyjne i czytelne strony, które przyciągają uwagę użytkowników.
